- Data in python - text (**str**), numbers, tabular data, images and sounds
  - how are images stored? - each pixel corresponds to a number that inturn relates to a color(RGB).
  - sounds stored? - using very rapid variations in air pressure, and its by storing these diff. measurements of air pressure as bunch of numbers that's how a computer stores sound.
  - using triple quotation marks - store a multiline string.
  - To check the data type, you can use the type() function. Functions are commands.   
  - **Int** - Integers are the +ve and -ve whole numbers, like 42, 100, -9, and 0. If there are no quotes around the number Python assumes this is numerical data, and if there's no decimal place on the number, it interprets it as an integer.
  - **Float** - If there is a decimal point - then its float -  data type used to store floating point numbers. Floating point numbers are positive and negative numbers that include a decimal place, like 3.14, 2.99, and -0.003.
- Python works great for arithmetic operations, its possible to perform advanced math, like computing the compound interest after 10 years at a rate of 5%.
  - ```
    print(1.05 ** 10)
    ```
    - in python, ** means "to the power of. ^ is a different operation (XOR - exclusive OR).
      - XOR - compares bits and gives 1 when exactly one of the two bits is 1, otherwise 0. Truth table for two bits:
        0 XOR 0 = 0,
        0 XOR 1 = 1,
        1 XOR 0 = 1,
        1 XOR 1 = 0,

        Example with 5 ^ 3:
        5 in binary = 101,
        3 in binary = 011,
        XOR them bit by bit: 101 XOR 011 = 110,
        110 in binary = 6, so 5 ^ 3 = 6   --- Common uses: toggling bits, simple checks for parity, and some algorithms that cancel duplicates.
